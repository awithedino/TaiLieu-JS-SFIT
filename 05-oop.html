Lập Trình Hướng Đối Tượng (OOP) trong JavaScript

Kiến thức về lập trình hướng đối tượng (OOP) trong JavaScript, bao gồm:

  • Đối tượng: Tạo, truy cập và thay đổi thuộc tính.
  • Kế thừa: Sử dụng prototype để kế thừa.
  • Lớp: Cách tạo lớp và sử dụng từ khóa class trong ES6.

1. Đối tượng

Trong JavaScript, đối tượng là một tập hợp các thuộc tính và phương thức. Bạn có thể tạo, truy cập và thay đổi thuộc tính của đối tượng như sau:

1.1 Tạo đối tượng


let person = {
    name: "Nguyễn Văn A",
    age: 30,
    greet: function() {
        console.log("Xin chào, tôi là " + this.name);
    }
};

1.2 Truy cập và thay đổi thuộc tính


// Truy cập thuộc tính
console.log(person.name); // Nguyễn Văn A

// Thay đổi thuộc tính
person.age = 31;
console.log(person.age); // 31

// Gọi phương thức
person.greet(); // Xin chào, tôi là Nguyễn Văn A

2. Kế thừa

Kế thừa cho phép một đối tượng nhận các thuộc tính và phương thức từ một đối tượng khác. Trong JavaScript, bạn có thể sử dụng prototype để thực hiện kế thừa.

2.1 Kế thừa bằng prototype


function Animal(name) {
    this.name = name;
}

Animal.prototype.speak = function() {
    console.log(this.name + " phát ra tiếng!");
};

function Dog(name) {
    Animal.call(this, name); // Gọi hàm khởi tạo của lớp cha
}

// Kế thừa từ Animal
Dog.prototype = Object.create(Animal.prototype);
Dog.prototype.constructor = Dog;

let dog = new Dog("Chó");
dog.speak(); // Chó phát ra tiếng!

3. Lớp

Từ ES6, JavaScript hỗ trợ từ khóa class để định nghĩa lớp, giúp đơn giản hóa việc tạo lớp và kế thừa.

3.1 Tạo lớp


class Animal {
    constructor(name) {
        this.name = name;
    }

    speak() {
        console.log(this.name + " phát ra tiếng!");
    }
}

class Dog extends Animal {
    constructor(name) {
        super(name); // Gọi hàm khởi tạo của lớp cha
    }

    speak() {
        console.log(this.name + " sủa!");
    }
}

let dog = new Dog("Chó");
dog.speak(); // Chó sủa!

3.2 Phương thức tĩnh


class MathUtil {
    static add(a, b) {
        return a + b;
    }
}

// Sử dụng phương thức tĩnh
console.log(MathUtil.add(5, 10)); // 15
