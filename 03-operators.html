Các toán tử trong JavaScript

Bài giảng này sẽ cung cấp kiến thức về các toán tử trong JavaScript, bao gồm:

  • Toán tử số học
  • Toán tử gán
  • Toán tử chuỗi
  • Toán tử logic
  • Toán tử bitwise
  • Toán tử so sánh

1. Toán tử số học

Các toán tử số học dùng để thực hiện các phép toán trên số:


let a = 10;
let b = 5;

// Cộng
let sum = a + b;  // 15

// Trừ
let difference = a - b;  // 5

// Nhân
let product = a * b;  // 50

// Chia
let quotient = a / b;  // 2

// Chia lấy phần dư
let remainder = a % b;  // 0

2. Toán tử gán

Toán tử gán được sử dụng để gán giá trị cho biến:


let x = 10;
x += 5; // Tương đương với x = x + 5
// Giá trị của x sau khi gán: 15

3. Toán tử chuỗi

Toán tử chuỗi dùng để nối các chuỗi lại với nhau:


let firstName = "Nguyễn";
let lastName = "Văn A";
let fullName = firstName + " " + lastName; // Nguyễn Văn A

4. Toán tử logic

Toán tử logic được sử dụng để kết hợp các biểu thức boolean:


let isAdult = true;
let hasLicense = false;

// AND
let canDrive = isAdult && hasLicense; // false

// OR
let canVote = isAdult || hasLicense; // true

// NOT
let isMinor = !isAdult; // false

5. Toán tử bitwise

Toán tử bitwise được sử dụng để thao tác với các bit:


let num1 = 5;  // 0101
let num2 = 3;  // 0011

// AND bitwise
let andResult = num1 & num2; // 1 (0001)

// OR bitwise
let orResult = num1 | num2; // 7 (0111)

// XOR bitwise
let xorResult = num1 ^ num2; // 6 (0110)

// NOT bitwise
let notResult = ~num1; // -6

6. Toán tử so sánh

Toán tử so sánh được sử dụng để so sánh hai giá trị:


let x = 10;
let y = 20;

// Bằng
let isEqual = (x === y); // false

// Khác
let isNotEqual = (x !== y); // true

// Lớn hơn
let isGreater = (x > y); // false

// Nhỏ hơn
let isLess = (x < y); // true

// Lớn hơn hoặc bằng
let isGreaterOrEqual = (x >= y); // false

// Nhỏ hơn hoặc bằng
let isLessOrEqual = (x <= y); // true
