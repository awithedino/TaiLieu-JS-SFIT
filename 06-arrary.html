Mảng và Các Phương Thức của Mảng trong JavaScript

Kiến thức về mảng trong JavaScript, bao gồm cách khởi tạo, thao tác với mảng, và các phương thức của mảng.

1. Khởi tạo và thao tác với mảng

1.1 Khởi tạo mảng


let fruits = ["Táo", "Chuối", "Cam"]; // Khởi tạo mảng

1.2 Thêm phần tử vào mảng


// Thêm vào cuối mảng
fruits.push("Nho"); // ["Táo", "Chuối", "Cam", "Nho"]

// Thêm vào đầu mảng
fruits.unshift("Dưa Hấu"); // ["Dưa Hấu", "Táo", "Chuối", "Cam", "Nho"]

1.3 Xóa phần tử khỏi mảng


// Xóa phần tử cuối mảng
fruits.pop(); // ["Dưa Hấu", "Táo", "Chuối", "Cam"]

// Xóa phần tử đầu mảng
fruits.shift(); // ["Táo", "Chuối", "Cam"]

1.4 Tìm kiếm phần tử trong mảng


let index = fruits.indexOf("Chuối"); // 1
let exists = fruits.includes("Cam"); // true

2. Các Phương Thức của Mảng

Dưới đây là các phương thức thông dụng của mảng trong JavaScript:

2.1. Phương thức map()

Phương thức map() tạo một mảng mới bằng cách thực hiện một hàm trên mỗi phần tử của mảng cũ.


let numbers = [1, 2, 3, 4, 5];
let doubled = numbers.map(num => num * 2); // [2, 4, 6, 8, 10]

2.2. Phương thức filter()

Phương thức filter() tạo một mảng mới chứa các phần tử thoả mãn điều kiện trong hàm kiểm tra.


let evenNumbers = numbers.filter(num => num % 2 === 0); // [2, 4]

2.3. Phương thức reduce()

Phương thức reduce() thực hiện một hàm trên mỗi phần tử của mảng, gộp lại thành một giá trị duy nhất.


let sum = numbers.reduce((accumulator, currentValue) => accumulator + currentValue, 0); // 15

2.4. Phương thức forEach()

Phương thức forEach() thực hiện một hàm trên mỗi phần tử của mảng nhưng không tạo ra mảng mới.


numbers.forEach(num => console.log(num)); // In ra từng số

2.5. Phương thức some()

Phương thức some() kiểm tra xem có ít nhất một phần tử trong mảng thỏa mãn điều kiện không.


let hasEvenNumber = numbers.some(num => num % 2 === 0); // true

2.6. Phương thức every()

Phương thức every() kiểm tra xem tất cả các phần tử trong mảng có thỏa mãn điều kiện không.


let allPositive = numbers.every(num => num > 0); // true

2.7. Phương thức find()

Phương thức find() trả về phần tử đầu tiên thỏa mãn điều kiện trong hàm kiểm tra.


let firstEven = numbers.find(num => num % 2 === 0); // 2
